Setup instructions:

1. Install Unity 2018.4.20f1

2. Make a new project

3. Drag/Drop each package in the package folder into the projects tab, import all as the dialogs come up.
 - VRCSDK3
 - DynamicBones
 - PoiToonShader

4. Copy the Scene folder into your Assets folder.
 - It's ok to totally replace the existing folder with the sample scene.

5. The avatars are located in:
 - Scenes/Virtualis/Guides/VirtualisGuides.unity
 - Scenes/Virtualis/Guides/VirtualisManikins.unity
